HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objets, such as interactive forms, may be embedded into the rendered page. It provides a means to creat structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and otehr items. HTML elemnts are delineated by tags, written using angle brackets.
